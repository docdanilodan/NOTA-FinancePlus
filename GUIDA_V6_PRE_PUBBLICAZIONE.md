# Guida rapida F+.Tech v6 pre-pubblicazione

## Dashboard
Pulsanti rapidi:
- Nuova Nota
- Nuova Pratica
- Smart Import
- Scarica Mail
- Anagrafica
- Scheda Azienda
- Documenti
- Report

## Mail
Quando scarichi le mail:
1. salva mail e allegati;
2. crea cartelle per periodo e mittente;
3. se non trova l'azienda, crea automaticamente un cliente;
4. registra gli allegati anche in Docs.

## Docs
Puoi:
- caricare documenti;
- vedere archivio;
- selezionare più documenti;
- inviarli via email con SMTP.

## Scheda Azienda
Mostra:
- anagrafica;
- documenti;
- mail;
- note;
- pratiche;
- PDF scheda azienda.

## Admin
Puoi creare utenti, cambiare password e scaricare backup.
