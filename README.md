# FinancePlus 360 Enterprise v1

Applicazione Streamlit per NOTA FinancePlus Cloud.

## Accesso iniziale
- Utente: admin
- Password: admin123

## File principale Streamlit
Usare uno dei due:
- app.py
- nota_webapp_google_drive.py

## Deploy
1. Caricare tutti i file nel repository GitHub.
2. Su Streamlit Cloud indicare come main file: `nota_webapp_google_drive.py` oppure `app.py`.
3. Fare Reboot app.

- Dashboard
- NOTA
- ANAGRAFICA
- FINANCE
- DOCS
