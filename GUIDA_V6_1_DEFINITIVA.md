# Guida F+.Tech v6.1 definitiva pre-pubblicazione

## Prima operazione
Dopo il login con admin/admin123 vai in:
Admin > Cambio password

## Dashboard
Usa i pulsanti rapidi per:
- Nuova Nota
- Nuova Pratica
- Smart Import
- Scarica Mail
- Anagrafica
- Scheda Azienda
- Documenti
- Report

## Mail
Il pulsante "Scarica tutte le mail nuove" parte automaticamente dalla data successiva all'ultimo scarico.
Se l'azienda non esiste, la crea automaticamente.

## Documenti
Ogni allegato mail viene salvato anche in Docs.
Puoi selezionare più documenti e inviarli via email.

## Scheda Azienda
Contiene tutto il fascicolo cliente:
- anagrafica
- documenti
- mail
- note
- pratiche
- scadenze
- banche
- rating
- PDF completo

## Report
I PDF hanno copertina, logo, riepilogo, tabelle e firma.
